算术运算符

算术运算

  • + 加,addition
  • - 减,subtraction
  • * 乘, multiplication
  • / 除,division
  • % 取余（相除后的余数）,modulo
  • ** 取幂（注意 ^ 并不执行该运算，你可能在其他语言中见过这种情形）,exponent , exponentiation
  • // 相除后向下取整到最接近的整数, integer division
  • ‘*’
