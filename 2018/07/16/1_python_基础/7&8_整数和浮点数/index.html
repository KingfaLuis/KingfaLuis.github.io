通常，我们需要注意两种类型的错误

  • 异常
  • 语法错误

异常是代码运行时发生的问题，而语法错误是 Python 在运行代码之前检查代码时发现的问题。要了解详情，请参阅关于错误和异常的 Python 教程页面。
